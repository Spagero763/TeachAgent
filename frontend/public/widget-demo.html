TeachAgent Widget Demo

This page embeds the TeachAgent widget. Look for the green button in the bottom-right corner — click it to ask anything about Celo.

Add it to any site with one line:

<script src="https://teach-agent.vercel.app/widget.js"></script>

Optional config:

<script>
  window.TeachAgentConfig = {
    position: "bottom-right", // or "bottom-left"
    color: "#35D07F",
    label: "Ask about Celo"
  }
</script>
